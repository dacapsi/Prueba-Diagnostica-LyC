## Estructura del Repositorio
*   `/Problema1`: Analizador léxico de expresiones aritméticas.
*   `/Problema2`: Validador de notación FEN para ajedrez.
*   `/Problema3`: Verificador de la Conjetura de Collatz.
*   `instalar.bat`: Script de configuración automática para windowns.
*   `instalador.sh`: Script de configuración automítica para Linux/Mac

## Instalación y Ejecución
1.  **Instalar dependencias:** Ejecute el script `instalar.bat` (Windows) o `bash instalar.sh` (Linux).
2.  **Ejecutar Problemas:**
    *   `python Problema1/analizador.py`
    *   `python Problema2/validador_fen.py`
    *   `python Problema3/collatz.py`
